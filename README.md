### Описание проекта

Система представляет собой программное решение на Python для автоматизации процесса резервного копирования данных. Программа обеспечивает создание, управление и восстановление резервных копий с поддержкой полного и инкрементного копирования, а также включает функции шифрования и сжатия данных для обеспечения безопасности и экономии дискового пространства.

**Ключевые возможности:**

- Создание полных резервных копий всех данных из указанных директорий
    
- Поддержка инкрементного копирования, сохраняющего только измененные файлы
    
- Автоматическое шифрование резервных копий с использованием алгоритма Fernet
    
- Сжатие данных в формате ZIP для экономии места
    
- Автоматическое планирование регулярного резервного копирования
    
- Простое восстановление данных из резервных копий
    
- Просмотр истории и списка всех созданных копий
    

### Установка и требования

**Необходимое ПО:**

- Python 3.13 или выше

**Установка зависимостей:**
pip install cryptography schedule

### Использование системы

#### Основные команды:

1. **Полное резервное копирование:**
    python .\main.py full

Создает полную резервную копию всех файлов из директории `data`

2. **Инкрементное резервное копирование:**
    python .\main.py incremental

Создает резервную копию только тех файлов, которые изменились после последнего бэкапа

3. **Восстановление данных:**
    python .\main.py restore 

Восстанавливает данные из указанного файла резервной копии в директорию `restored`

4. **Просмотр списка копий:**
    python .\main.py list

Отображает список всех доступных резервных копий

5. **Запуск по расписанию:**
    python .\main.py schedule

Запускает автоматическое инкрементное копирование каждые 24 часа

6. **Демонстрационный режим:**
    python .\main.py demo
Запускает автоматическую демонстрацию работы всех функций системы
 7. **Подставка -dir** 
 делает конкретный бэкап файла или папки
 python .\main -dir (путь) (команда)
 пример: python .main -dir ("D:\Downloads\проект") full
